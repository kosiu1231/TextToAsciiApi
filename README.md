# Text To Ascii Api

Simple app generating ascii art based on the given prompt.

English prompts only!

## Demo

https://text-to-ascii-nine.vercel.app/

## Tech Stack

* .NET 8
* HTML/CSS/JS
* Pexels API
* Azure
* Vercel
